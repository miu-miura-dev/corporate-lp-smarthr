## 📂 GitHub

---

## 📝 概要

React（Vite + TypeScript）を使用して、
コーポレート採用LPを制作しました。

Figmaデザインを想定し、
コンポーネント設計・レスポンシブ対応・フォーム制御まで実装しています。

---

## 🎯 制作目的

- Figma → React実装力の証明
- コンポーネント設計力のアピール
- フォーム制御・状態管理の実装
- API分離構成の練習

---

## 🛠 使用技術

- React
- TypeScript
- Vite
- CSS
- useState
- mock API（非同期処理）
- コンポーネント分割設計

---

## 💡 実装ポイント

- 各セクションをコンポーネント単位で分割
- 再利用可能なButtonコンポーネント作成
- ハンバーガーメニュー（useState制御）
- フォームバリデーション
- 非同期mock API実装
- レスポンシブ対応（768px以下）

---

## 📌 工夫した点

- UIの共通化（Button）
- APIロジックを`api/`ディレクトリに分離
- hooksを使用した状態管理
- モバイル表示でのナビゲーション制御
